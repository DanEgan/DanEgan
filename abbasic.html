----HTML Images----

HTML images are defined with the < img > tag.

These have Five (5) parts or attributes:

img (this introduces an image file)

src="" (this is the surce of the file) after this

alt="" (this is what is read [by screen readers] if the file can't be displayed)

width="" (this tells how wide the image is)

height="" (this tells how heigh the image is)

< img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">

----HTML Lists----

HTML lists are defined with the < ul > (unordered/bullet list)

or the < ol > (ordered/numbered list) tag,

followed by < li > tags (list items):

An Unordered HTML List

  • Coffee
  • Tea
  • Milk

An Ordered HTML List

  1. Coffee
  2. Tea
  3. Milk
