The start attribute

Top
Control List Counting

By default, an ordered list will start counting from 1. Use the start attribute to start counting from a specified number:

  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk
 
    
 This is what it really says:

    < ol start="50">
      < li>Coffee< /li>
      < li>Tea< /li>
       < li>Milk< /li>
    < /ol>

    < ol type="I" start="50">
      < li>Coffee< /li>
      < li>Tea< /li>
      < li>Milk< /li>
    < /ol>
