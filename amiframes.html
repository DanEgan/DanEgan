Iframe Syntax Top


  An HTML iframe is defined with the < iframe> tag:

  < iframe src="URL">< /iframe>
  The src attribute specifies the URL (web address) of the inline frame page.


Iframe - Set Height and Width Top


    < iframe src="demo_iframe.htm" style="height:50px;width:50px;">< /iframe>

Iframe - Remove the Border Top


      < iframe src="demo_iframe.htm" style="border:none;">< /iframe>

Iframe - Change the Size, Style, and Border Top


        < iframe src="demo_iframe.htm" style="border:2px solid red;">< /iframe>

Iframe - Target for a LinkTop

When the target of a link matches the name of an iframe, the link will open in the iframe.

W3Schools.com

 

      < iframe height="300px" width="100%" src="demo_iframe.htm" name="iframe_a">< /iframe>

      < p>< a href="https://www.w3schools.com" target="iframe_a">W3Schools.com< /a>< /p>
